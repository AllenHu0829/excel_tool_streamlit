# 标题区域背景图片说明

## 如何使用

程序已经支持在标题区域显示背景图片。只需要将你的猫咪图片放在程序目录下即可。

## 支持的图片格式

- PNG (.png)
- JPG/JPEG (.jpg, .jpeg)

## 图片命名

请将猫咪图片命名为以下任一名称之一：
- `cat_bg.png` 或 `cat_bg.jpg` （推荐）
- `header_bg.png` 或 `header_bg.jpg`

程序会按顺序查找这些文件。

## 图片位置

将图片文件放在与 `excel_tool_gui.py` 相同的目录下，例如：
```
excel data/
├── excel_tool_gui.py
├── cat_bg.png        ← 放在这里
└── ...
```

## 图片尺寸

- 图片会自动调整到 850x140 像素以适应标题区域
- 建议使用宽高比约为 6:1 的图片
- 程序会自动裁剪和调整，无需手动处理尺寸

## 注意事项

1. 如果没有找到图片文件，程序会使用白色背景（默认）
2. 为了文字清晰度，程序会在图片上添加半透明白色遮罩
3. 如果图片加载失败，程序会自动使用默认背景色

## 示例

将你的猫咪图片命名为 `cat_bg.png` 并放在程序目录下，重启程序即可看到效果。
